Traveling Salesman Problem (TSP) with Genetic Algorithm

The following code is designed to solve the Traveling Salesman Problem (TSP) using a Genetic Algorithm and to visualize the cities on a map using Leaflet.

This code is based on the implementations of the Genetic Algorithm provided by the Project Spot, which can be found at the following link: Genetic Algorithm Tutorial.

Additionally, it utilizes the Haversine formula to calculate distances, following the code example by Parzibyte, available at: Haversine Formula in JavaScript.
